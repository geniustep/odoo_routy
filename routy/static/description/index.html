Routy - Smart Delivery System

A comprehensive delivery and logistics management system for Odoo 18

Features

Service Request Management

  • Create and manage delivery requests
  • Track pickup and delivery addresses
  • Assign drivers to requests
  • Monitor request status

Parcel Tracking

  • Generate unique tracking numbers
  • Track parcel status in real-time
  • Proof of Delivery (POD) with signature and photo
  • Physical dimensions and weight tracking

Driver Job Management

  • Create pickup and delivery jobs
  • Assign jobs to drivers
  • GPS tracking integration
  • Job status workflow

Route Planning

  • Create daily route plans for drivers
  • Route optimization
  • Calendar view for scheduling
  • Performance tracking

Hub & Linehaul Management

  • Manage distribution hubs
  • Inter-city transport planning
  • Capacity management
  • Cost tracking

Payment Collection

  • Cash on Delivery (COD) tracking
  • Payment reconciliation
  • Multiple payment methods
  • Financial reports

Mobile API

  • RESTful API for driver mobile apps
  • GPS location updates
  • Job management via API
  • Digital proof of delivery

Incident Management

  • Report and track incidents
  • Severity and priority levels
  • Resolution workflow
  • Preventive measures documentation

User Roles

  • Driver: View assigned jobs, update GPS, mark deliveries
  • Dispatcher: Create requests, assign drivers, manage operations
  • Manager: Full system access, reports, and configuration
